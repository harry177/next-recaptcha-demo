Before running app create required keys for tested cases and add them to env file using template:

# v2 (checkbox + invisible)

NEXT_PUBLIC_RECAPTCHA_V2_CHECKBOX_SITE_KEY=...
RECAPTCHA_V2_CHECKBOX_SECRET_KEY=...

NEXT_PUBLIC_RECAPTCHA_V2_INVISIBLE_SITE_KEY=...
RECAPTCHA_V2_INVISIBLE_SECRET_KEY=...

# v3

NEXT_PUBLIC_RECAPTCHA_V3_SITE_KEY=...
RECAPTCHA_V3_SECRET_KEY=...
